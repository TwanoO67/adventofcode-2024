# Advent Of Code 2024

## Start A New Day

```
npx hygen new day --day 1
```

## Run the tests

```
npm run test
```

## Run the script

Run all the day

```
node --experimental-strip-types --experimental-transform-types index.ts 1
```

Run a specific part of a day

```
node --experimental-strip-types --experimental-transform-types index.ts 1 1
```
